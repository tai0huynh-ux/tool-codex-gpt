# Uninstall

1. Close Codex Context Bridge.
2. Remove the desktop app through Windows **Installed apps**. This removes the packaged Native Messaging registration and installed payload.
3. Remove the unpacked extension from `edge://extensions`.

Uninstall does not imply permission to delete user data. Keep the Electron application-data directory when project history is needed. To erase it, first locate the exact Codex Context Bridge directory, back up anything required, and delete only that confirmed directory after the app is closed.
